# studyNotes

> Inspired by [The Illustrated GRPO](https://github.com/abderrahmanskiredj/the-illustrated-grpo), I aim to systematically organize my learning and reflections on large language models (LLMs) and cutting-edge AI directions into a series of articles. At the same time, drawing on the papers I have read, I will expand from fundamental principles to more advanced research and practices within the `12-paper-notes` directory, creating a learning repository that serves both as a personal synthesis and a useful reference for others.
